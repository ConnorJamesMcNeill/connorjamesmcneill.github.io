### The System

The brief for this 4th year project was to simulate the logic controls of an aquarium water monitoring system. The system needed to handle temperature sensing, turbidity monitoring, a coolant pump, indicator lights, a timer, and a manual pump control. None of that is particularly exotic in a real aquarium controller, but the point of the project was the control logic, not the hardware. Everything was implemented on a breadboard with an Elegoo Arduino Mega 2560, using simple sensors and switches as stand-ins for the real-world actuators and instrumentation the system would eventually drive. The code was written in Arduino with an OpenPLC plugin to structure the ladder logic.
